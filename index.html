HTML


Ideas Generales


  • El HTML es un documento de texto que estrucuctura una pagina web

  • Su estructura basica contiene varias etiquetas como < !DOCTYPE html>, < html>, seguidas por un < head> y < body>.

  • HTML tiene como regla tener dos etiquetas, una para abrir nuestro etiqueta y otra para cerrar, la etiqueta abrir se ve de esta manera < html> y la etiqueta de cierra se ve asi < /html>

  • Entre el inicio y el final de la etiqueta de contener el contenido por ejemplo: < h1>Titulo< /h1>

  • Los simbolos menor que < y mayor que > son muy necesarias para utilizar la funcion de la etiqueta.

  • Algo fundamental una vez creado tu archivo HTML es necesario utilizar el nombre "index.html" ya que es una eficiente para que cualquier navegador lo encuentre

  • HTML, que significa HyperText Markup Language

Apuntes de Clase


El HTML es una estrucutra de texto para el formato de paginas web, el cual lleva una estrucutura que debe ser formada de manera correcta el cual el index empieza con DOCTYPE html indentificando que el tipo de texto es un HTML, seguida por un header el cual se usa para insertar algunos enlaces externoal index, seguido por la etiqueta body que se usa para el cuerpo de la pagina y con el fin de la pagina se usa para el footer el cierra de la pagina.



HTML, que significa HyperText Markup Language (Lenguaje de Marcado de Hipertexto), es el lenguaje estándar que se utiliza para estructurar y presentar contenido en las páginas web. La estructura HTML es como el esqueleto de una página web. Comienza con una declaración del tipo de documento (DOCTYPE) que indica al navegador la versión de HTML que se está utilizando. A continuación, se encuentran las etiquetas < html>, que engloban todo el contenido de la página. Dentro de estas, se distinguen dos secciones principales: < head>, que contiene información meta como el título, codificación de caracteres y enlaces a hojas de estilo, y < body>, donde se coloca todo el contenido visible de la página, como encabezados, párrafos, imágenes, enlaces y otros elementos. Esta estructura jerárquica, definida por etiquetas de apertura y cierre, permite a los navegadores interpretar y mostrar el contenido de una manera organizada y coherente..



TEXTO HTML


Ideas Generales


  • Existe varias etiquetas para modificar los estilos del texto para darle un estilo unico y diferente.

  • Las principales etiquetas son:< h1>,< h2>, < h3>, < h4>,< h5>, < h6>, < p>, < li> y < ul>.

  • Los encabezados tiene etiquetas como: < h1>,< h2>, < h3>, < h4>,< h5>, < h6> y cada numero depende el nivel de jerarquia de cada uno.

  • Los parrafos tiene la etiqueta < p> para identificar un parrafo.

  • La etiqueta < li> se usa para hacer una lista ordenada, con finalidad de tener un orden especifico.

  • La etiqueta < ul> sirve para hacer una lista desordena donde no importa el orden, por ejemplo una lista de compras.

  • Existe varias etiquetas para aplicar estilos a las etiquetas de texto en las que resaltan: < b> Negritas, < mark> Texto marcado, < small> Texto pequeño, entre otras.

RESUMEN

Las etiquetas HTML como < h1>, < h2>, < p>, < ul>, < li>, < b>, < mark>, y < small> permiten dar formato y estructura a los textos en una página web. Los encabezados < h1> a < h6> organizan el contenido por jerarquía, mientras que < p> define párrafos. Las listas < ul> y < li> se utilizan para enumerar elementos, y las etiquetas de formato como < b> y < mark> agregan énfasis visual al texto.

ENLACES HTML


Ideas Generales


  • Los enlances nos permiten navegar entre pagina o paginas dentro de sitio web

  • Los enlaces HTML, también conocidos como hipervínculos, son elementos fundamentales en la creación de páginas web. Estos permiten a los usuarios navegar de una página a otra, ya sea dentro del mismo sitio web o hacia un sitio externo.

  • Se usan las etiqueta < a> que indican el inicio y final de la etiqueta, href para especificar la ruta de destino y el texto de enlaces es texto que mostrara en el enlace

  • Enlaces internos: Conectan diferentes páginas dentro del mismo sitio web.

  • Enlaces externos: Conectan a páginas en otros sitios web.

  • Enlaces a archivos: Permiten descargar archivos

  • Enlaces a secciones de la misma página: Utilizan el símbolo # seguido de un identificador para enlazar a una sección específica dentro de la misma página. Otros atributos útiles:

RESUMEN

CSS


Ideas Generales


  • CSS (Cascading Style Sheets) es un lenguaje que define la apariencia de un documento escrito.

  • Sirve para agregar estilo o diseño por ejemplo, fuentes, colores, espaciado a documentos web.

  • Hay tres tres formas de poner estilos en CSS en el HTML: Incrustada, En linea y Externa.

  • Hoja incrustada: esta hoja se puede incorporar dentro del index en la etiqueta headear y se incorpora con la etiqueta < stylé>

  • Hoja en linea: Puede usarse estilos en línea mediante el atributo style. • El atributo style puede ser aplicado a cualquier elemento HTML (incluyendo en < body>).

  • Hoja externa: Una hoja de estilo externa puede ser enlazada a un documento HTML mediante el elemento < link> de HTML

  • El codigo CSS se compone de reglas con un selector "#nombre", propiedad "font-family" y un valor :arial;

  • Se usan {} y se utilizan : para cerrar la propiedad y ; para el valor

Apuntes de Clase

PROCESO DE DISEÑO


Ideas Generales


  • El proceso de diseño tiene varias fases para llevar acabo un proyecto web, planificacion, benchmarking, modelado de usuario y diseño conceptual.

  • La planificacion es la parte en donde se desarrolla el para que de la pagina, sus funciones, es mas que un analisis completo para el desarrolo de un proyecto

  • Benchmarking es hacer un sondeo de competidores con relacion de nuestra pagina web, en donde vemos como ellos han realizado su pagina web, en donde hacemos un analisis, para saber que aspecto mejorar y que aspectos podemos implentar

  • Modelado de usuario es la seccion en la que hacemos un analisis de las personas que son de nuestro mercado, indentificando las necesidades de los usuarios a los que va dirigidos para mejorar el enfoque de la pagina.

  • Diseño conceptual es una sketch de como queremos estrucutrar nuestra pagina web, para tener una orientacion de como queremos que esta se vista y para completar, es fundamental porque ya de ahi tendremos que hacer nuestro diseño final en cual este contrendra las imagenes y estilos y colores.

Apuntes de Clase


El proceso de diseño es muy importante a la hora de realizar un proyecto web, ya que este contendra toda la informacion recopilada para que el sitio web sea competitivo dentro del mercado, este constanta de varias fases para que el margen de error sea el minimo para la eficiencia a la hora de ejecutar el proyecto, estas fases recopilan informacion del usuarios, informacion de otros competidores y nos da una vision de como competir contra otras empresas desarroladoras web.



RESUMEN

El proceso de diseño implica una serie de etapas interconectadas que comienza con el benchmarking, donde se estudian los mejores ejemplos de la industria para identificar oportunidades de mejora. A continuación, se realiza un modelado detallado de los usuarios para comprender sus necesidades y comportamientos. Con esta información, se elabora un plan de diseño que define los objetivos, estrategias y tareas necesarias para crear una solución que satisfaga las necesidades de los usuarios y se destaque en el mercado. Este proceso iterativo garantiza que el diseño final sea relevante, útil y exitoso.
